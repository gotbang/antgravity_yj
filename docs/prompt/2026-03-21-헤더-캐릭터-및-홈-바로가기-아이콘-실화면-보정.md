# 헤더 캐릭터 및 홈 바로가기 아이콘 실화면 보정 프롬프트 기록

- 작성시각: 2026-03-21 01:12:00 +09:00

## 해결하고자 한 문제

localhost에 띄운 실제 화면 기준으로 헤더의 어색한 부분을 직접 점검하고, 업로드한 이미지처럼 개미 캐릭터 중심의 브랜드 헤더와 홈 바로가기 3개 아이콘으로 바꾸는 것.

## 사용한 프롬프트 요약

- `http://localhost:5174/ 띄어놨으니 직접 테스트. 특히 헤더부분 부자연스러운점. 그리고 업로딩한 이미지의 헤더부분 개미캐릭터로 헤더부분 개미 변경. 오늘판단, 종목분석, 개미의 일기부분도 변경.`

## 해결된 것

- 실제 홈/감정 화면 캡처 기반 점검
- 공통 헤더 개미 캐릭터 일러스트 적용
- 홈 3개 바로가기 일러스트 아이콘 교체
- 헤더 여백, 네비 칩, 타이포 보정
- 테스트/타입체크/린트 완료

## 아직 안 된 것

- 업로드 원본과 1:1 동일한 캐릭터 자산 반영
