# Figma site 기반 MVP 구현 프롬프트 기록

- 작성시각: 2026-03-20 20:22:03

## 해결하고자 한 문제

피그마 링크와 공개된 Figma site를 기준으로 비슷한 정보 구조의 웹앱을 먼저 구현하고, 이후 디자인을 한 단계 업그레이드할 수 있는 기반을 만드는 것.

## 사용한 프롬프트 요약

- 공개된 Figma site를 분석해서 라우트와 카드 구조를 파악해줘.
- 빈 워크스페이스에서 React + Vite + TypeScript로 MVP를 구현해줘.
- 홈, 종목 분석, 시장 감정 3개 화면을 만들고 관심 종목 로컬 저장을 붙여줘.
- 테스트, 린트, 타입체크, 빌드까지 통과시켜줘.

## 해결된 것

- 레퍼런스 구조 분석
- MVP 라우트 구현
- 테스트/검증 통과
- 문서화

## 아직 안 된 것

- 원본 Figma 노드 기준 1:1 픽셀 검증
- 실제 API 연동
- 후속 디자인 업그레이드
