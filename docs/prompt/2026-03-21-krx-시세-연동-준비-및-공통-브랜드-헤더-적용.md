# KRX 시세 연동 준비 및 공통 브랜드 헤더 적용 프롬프트 기록

- 작성시각: 2026-03-21 01:00:00 +09:00

## 해결하고자 한 문제

KRX OPEN API 키가 발급되면 바로 가격이 보이게 준비하고, 업로드한 이미지 느낌의 브랜드 헤더를 모든 페이지 상단에 공통으로 적용하면서 첫 화면으로 돌아가는 네비게이션도 넣는 것.

## 사용한 프롬프트 요약

- `krx open api는 발급대기중이야. 발급받아서 키값을 넣으면 바로 보여지도록 해주고 실제 제대로 동작하는지 테스트.`
- `이어서 해줘. 그리고 업로딩한 이미지부분을 첫화면의 개미캐릭터로 바꿔줘. 모든페이지 동일. 그리고 저화면에 네비게이션을 넣어 클릭하면 첫화면으로 가게끔 수정`

## 해결된 것

- KRX 시세 병합 코드와 테스트 추가
- KRX 미연동 fallback 유지
- 공통 브랜드 헤더/네비게이션 적용
- 홈 이동 동선 추가
- 테스트/타입체크/린트/빌드 완료

## 아직 안 된 것

- 실제 KRX 키 기반 외부 호출 검증
- KRX 승인 서비스 URL이 기본값과 다를 때 최종 확인
- 업로드 이미지 원본 파일 직접 반영
