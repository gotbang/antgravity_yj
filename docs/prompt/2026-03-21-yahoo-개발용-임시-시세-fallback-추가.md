# Yahoo 개인용 시세 fallback 추가 프롬프트 기록

- 작성시각: 2026-03-21 01:50:00 +09:00

## 해결하고자 한 문제

KRX OPEN API 키가 발급되기 전까지 개인용 배포와 로컬 화면에서도 시세가 보이게 만들고 싶었음.

## 사용한 프롬프트 요약

- `yfinance로 KRX OPEN API 키가 발급전까지 대체해서 시세가 나오게 하고 다른부분도 실제데이터로 변경.`

## 해결된 것

- 개인용 배포/로컬용 Yahoo 시세 fallback 추가
- 테스트 추가
- `.env.local` 스위치 활성화

## 아직 안 된 것

- 개인용 범위를 넘는 정식 시세 소스 확정
