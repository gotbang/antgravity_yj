# Figma 원본 콘텐츠 기반 화면 업그레이드

- 작성시각: 2026-03-20
- 해결하고자 한 문제: 디자인 스크린샷만 따라가면 정보 구조가 틀어질 수 있어서, Figma 원본 콘텐츠와 스크린샷 스타일을 동시에 반영할 구현 기준이 필요했음.

## 사용한 프롬프트 요약

- `이 기준서대로 실제 화면 업그레이드`
- `https://www.figma.com/design/... 여기 기준대로 구현중인건가`
- `Figma 원본으로 내용을 구현할건데 디자인만 design/스크린샷처럼 업그레이드 할려는 거였어`

## 해결된 것

- 구현 방향을 `Figma 원본 콘텐츠 + design 스크린샷 스타일`로 확정함.
- 실제 코드가 그 기준에 맞게 재구성되었고 검증까지 완료됨.

## 아직 안 된 것

- 세부 미세조정과 추가 1:1 다듬기는 후속 작업으로 남아 있음.
