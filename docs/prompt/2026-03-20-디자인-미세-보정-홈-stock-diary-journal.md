# 디자인 미세 보정 홈 stock diary journal 프롬프트 기록

- 작성시각: 2026-03-20 23:54:51 +09:00

## 해결하고자 한 문제

홈부터 저널까지 `design/` 스크린샷 기준의 디테일을 더 가깝게 만들고, 수정 뒤 바로 테스트까지 통과시키는 것.

## 사용한 프롬프트 요약

- `홈 -> stock -> stock/:symbol -> diary -> journal 순서로 디자인 미세 보정. 그리고 테스트`

## 해결된 것

- 화면별 디자인 미세 보정
- diary/journal 디테일 보정
- 테스트/타입체크/린트 완료

## 아직 안 된 것
