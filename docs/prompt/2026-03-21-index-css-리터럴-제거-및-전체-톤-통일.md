# index.css 리터럴 제거 및 전체 톤 통일 프롬프트 기록

- 작성시각: 2026-03-21 00:31:24 +09:00

## 해결하고자 한 문제

`index.css`의 남은 하드코딩 색상을 모두 토큰 기반으로 치환하고, 전체 페이지의 디자인 톤을 하나의 시스템으로 정리하는 것.

## 사용한 프롬프트 요약

- `index.css 남은 리터럴 전량 치환`
- `홈/stock/detail/diary/journal 전체 공통 톤 최종 정리`
- `끝나면 다시 테스트까지 돌려서 마무리`

## 해결된 것

- 토큰 사전 확장
- 공통 컴포넌트 리터럴 제거
- 전체 화면 톤 통일
- 테스트/타입체크/린트 완료

## 아직 안 된 것

- 토큰 파일 분리 리팩토링
- 페이지별 스타일 파일 분리
