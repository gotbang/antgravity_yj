## 해결하고자 한 문제

## 사용한 프롬프트 요약

- `이 규칙을 기준으로 실제 src/index.css를 토큰 구조로 재정리`
- `design-system-preview.html 같은 미리보기 페이지 만들어서 한 번에 검토 가능하게 하기`
