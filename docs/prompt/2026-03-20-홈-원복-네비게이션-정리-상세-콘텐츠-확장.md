# 홈 원복 네비게이션 정리 상세 콘텐츠 확장 프롬프트 기록

- 작성시각: 2026-03-20 23:35:25 +09:00

## 해결하고자 한 문제

Figma 원본과 공개 figma.site 기준의 정보 구조를 현재 프로젝트에 다시 맞추고, 디자인은 `design/` 폴더 스크린샷 톤을 유지한 채 홈/네비게이션/종목 상세의 누락된 콘텐츠를 복구하는 것.

## 사용한 프롬프트 요약

- `홈을 원본 기준으로 되돌리고, 개미의 일기 흐름을 /emotion으로 다시 맞춰줘`
- `종목 상세에 판단 근거, 리스크 평가, 전략 가이드, 데이터 믹스, AI 예측 따랐다면?를 추가해줘`
- `API 키가 없을 때도 샘플 콘텐츠가 먼저 보이게 해줘`
- `디자인은 design 폴더 안의 스크린샷 같은 디자인으로 유지해줘`

## 해결된 것

- 홈 구조 복구
- 메인 네비게이션 의미 정리
- 종목 상세 누락 섹션 추가
- 샘플 데이터 fallback 추가
- 테스트/타입체크/린트 검증 완료

## 아직 안 된 것

- Playwright 브라우저 최종 시각 검증
- 실데이터 주입 상태에서의 상세 QA
