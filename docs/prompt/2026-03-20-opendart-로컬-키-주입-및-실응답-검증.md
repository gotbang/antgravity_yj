# OpenDART 로컬 키 주입 및 실응답 검증 프롬프트 기록

- 작성시각: 2026-03-20 23:41:41 +09:00

## 해결하고자 한 문제

사용자가 제공한 OpenDART 키를 로컬에 안전하게 주입하고, 실제로 유효한 키인지와 공시 응답이 정상인지 확인하는 것.

## 사용한 프롬프트 요약

- `오픈 다트 키를 로컬에만 저장하고 응답 검증해줘`
- `키 값은 다시 노출하지 말고 실제 OpenDART 응답만 확인해줘`

## 해결된 것

- `.env.local` 저장
- 회사 정보 응답 검증
- 공시 목록 응답 검증

## 아직 안 된 것

- dev 서버를 통한 UI 단 실시간 치환 검증
