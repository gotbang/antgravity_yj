# 캐릭터 톤 강화 및 stock 실데이터 정리 프롬프트 기록

- 작성시각: 2026-03-21 00:06:17 +09:00

## 해결하고자 한 문제

첨부 레퍼런스 같은 둥근 폰트/캐릭터/분위기로 페이지 톤을 강화하고, 종목분석 화면에서 실제 OpenDART 데이터를 우선 노출하는 것.

## 사용한 프롬프트 요약

- `이런 폰트와 캐릭터, 느낌으로 전체페이지 수정 가능할까? 구조는 유지하면서`
- `종목분석에서 실제데이터로 나오게끔 하고싶다. 가격이나 이런게 맞지가 않다`

## 해결된 것

- 헤더와 배지 톤 강화
- stock 카드 실데이터 우선 구조화
- 가격 미연동 안내 추가
- 테스트/타입체크/린트 완료

## 아직 안 된 것

- 실시간 가격 API 연동
- 원본 캐릭터 에셋 반영
