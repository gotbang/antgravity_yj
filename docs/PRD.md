# 앤트그래비티 MVP PRD

## 개요

앤트그래비티는 개미 투자자가 시장 방향, 종목별 AI 예측, 시장 감정을 한 화면 흐름 안에서 읽을 수 있게 돕는 웹 앱이다.

이번 MVP는 공개된 Figma site의 정보 구조를 기준으로 다음 3개 화면을 구현한다.

1. 홈: 오늘의 시장 방향, 관심 종목 요약, 모델 신뢰도, 예측 히스토리
2. 종목 분석: 종목 검색, 인기 종목/관심 종목, 개별 종목 리포트
3. 시장 감정: 시장 분위기, 투자자 심리 지표, 뉴스 감정 분석

## 사용자 문제

- 초보 투자자는 시장 전체 분위기와 종목별 판단 근거를 따로따로 읽어야 해서 피로가 크다.
- 숫자만 많은 화면은 읽기 어렵고, 감정과 전략이 연결되지 않으면 행동으로 이어지기 어렵다.

## 목표

- 정보가 카드 단위로 빠르게 읽히는 구조를 만든다.
- 시장 방향, 종목 판단, 시장 감정을 서로 연결된 흐름으로 보여준다.
- 관심 종목을 브라우저 로컬 상태에 저장해서 다시 진입했을 때 이어볼 수 있게 한다.

## 비목표

- 실제 주문/매매 기능
- 실시간 시세 연동
- 로그인/회원 관리
- 위치 정보, GPS, 민감 개인정보 저장

## 핵심 기능

### 1. 홈

- 시장 상승/하락 확률 노출
- 변동 범위, 시장 리스크 표시
- 관심 종목 요약 카드
- 모델 신뢰도와 예측 히스토리 노출

### 2. 종목 분석

- 종목명 검색
- 인기 종목 목록
- 관심 종목 토글
- 종목 상세 리포트: 확률, 리스크, 판단 근거, 전략, 데이터 믹스

### 3. 시장 감정

- 공포/탐욕 성격의 감정 지표
- 투자자 수급 해석
- 뉴스 감정 비율 및 키워드
- 추천 대응 전략

## 데이터 원칙

- 모든 데이터는 MVP용 정적 샘플 데이터다.
- GPS, 좌표, 생체정보, 금융 계정 정보 같은 민감 정보는 저장하지 않는다.
- 관심 종목은 브라우저 `localStorage`에 종목명 문자열만 저장한다.

## 성공 기준

- `pnpm lint` 통과
- `pnpm typecheck` 통과
- `pnpm test` 통과
- `pnpm build` 통과
