# design 스크린샷 기준 홈 재보정

- 작성시각: 2026-03-20
- 해결하고자 한 문제: 현재 홈 화면이 `design/dashboard.png`와 구조적으로 너무 달라 보여서, 1단계 목표인 스크린샷 기준 1:1 재보정이 제대로 되지 않았음.

## 진행 내용

- `design/dashboard.png`, `design/analysis.png`, `design/prediction.png`, `design/diary.png`, `design/journal.png`를 다시 확인함.
- 특히 홈 화면의 브랜드 카드, 3개 진입 카드, 시장 감정 카드, 분위기 카드, 전략 pills 구조를 기준으로 잡음.
- `HomePage.tsx`를 다시 구성해서 `design/dashboard.png`와 더 비슷한 정보 위계로 정리함.
- `index.css`에 홈 전용 카드 스타일과 파스텔 톤을 추가함.
- 테스트 기대값도 홈 화면 구조 변경에 맞게 조정함.
- `pnpm test`, `pnpm typecheck`, `pnpm lint`를 실행함.

## 해결된 것

- 홈 화면이 스크린샷 기준과 더 비슷한 카드 구조로 정리됨.
- 브랜드 헤더와 3개 진입 카드가 시각적으로 더 가까워짐.
- 시장 감정/분위기/전략 영역이 스크린샷 톤에 맞춰짐.

## 아직 안 된 것

- 종목 분석/예측/일기/저널 화면도 같은 수준으로 추가 미세 보정 가능함.
- 실제 브라우저 캡처 대조 기반의 픽셀 단위 수정은 아직 안 함.
