# 디자인 미세 보정 홈 stock diary journal

- 작성시각: 2026-03-20 23:54:51 +09:00
- 해결하고자 한 문제: `design/` 스크린샷 기준의 큰 방향은 맞췄지만, 실제 모바일 레이아웃 디테일과 카드 밀도, 헤더 높이, 탭/칩/FAB 같은 반복 요소가 아직 덜 정렬돼 있었음.

## 진행 내용

- 홈 화면의 헤더, 3개 진입 카드, 카드 간격, 예측 히스토리 밀도를 다시 조정함.
- 종목 목록에서 상단 헤더 높이와 리스트 카드 섀도우를 보정하고, 검색창/탭 밀도를 스크린샷에 더 가깝게 맞춤.
- 종목 상세에서 상단 탭 높이와 메인 예측 카드의 중심 정렬을 보정함.
- 일기 화면에서 메인 아바타 톤, FAB 위치, 홈 인디케이터를 추가하고 카드 간격을 보정함.
- 저널 화면에서 카드 간격과 매매 일지 수익 금액 강조를 조정하고 예시 데이터도 스크린샷 톤에 맞게 다듬음.

## 해결된 것

- `홈 -> stock -> stock/:symbol -> diary -> journal` 순서의 미세 보정 완료
- 모바일 그리드 붕괴 규칙 정리
- 테스트/타입체크/린트 재검증 완료
