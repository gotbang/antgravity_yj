# dashboard 기준 헤더 캐릭터 및 공통 스타일 업그레이드

- 작성시각: 2026-03-21 01:35:00 +09:00
- 해결하고자 한 문제: `dashboard.png` 기준으로 헤더 개미 캐릭터와 전체 페이지 인상을 더 가깝게 맞추고, `MASTER.md`, `design-tokens.md` 기준을 실제 구현에 반영해서 모든 페이지 공통 톤을 끌어올려야 했음.

## 진행 내용

- 헤더 개미 캐릭터 SVG를 `dashboard.png`의 둥근 얼굴, 큰 눈, 원형 프로필 배지 인상에 더 가깝게 다시 그렸음.
- 공통 헤더 크기, 패딩, 타이포, 그림자를 `dashboard.png` 기준으로 재조정했음.
- 홈 3개 진입 카드의 크기, 아이콘 박스, 그림자, 텍스트 크기를 전체적으로 업그레이드했음.
- 공유 카드 그림자와 섹션 타이틀 크기를 조정해서 홈 외 다른 페이지도 같은 디자인 시스템 안에서 더 안정적으로 보이게 만들었음.
- `design/MASTER.md`와 `design/design-tokens.md`를 현재 구현과 최신 레퍼런스 기준으로 다시 정리했음.
- headless Chrome으로 `/`와 `/stock` 화면을 직접 캡처해서 실제 렌더 기준으로 확인했음.

## 해결된 것

- 헤더 개미 캐릭터가 기존보다 `dashboard.png` 인상에 더 가까워졌음.
- 모든 페이지가 같은 헤더 톤, 카드 볼륨, 그림자 강도를 공유하게 됨.
- 디자인 기준 문서와 실제 구현이 더 잘 맞게 정리됨.
- `pnpm test`, `pnpm typecheck`, `pnpm lint`, `pnpm build`를 모두 통과했음.

## 아직 안 된 것

- 업로드 원본 캐릭터를 그대로 자산화한 것은 아니고, 그 인상을 반영한 SVG 재구성본을 사용 중임.
- 필요하면 다음 단계에서 섹션별 세부 카드 배치도 `dashboard.png`에 더 가깝게 추가 보정할 수 있음.
