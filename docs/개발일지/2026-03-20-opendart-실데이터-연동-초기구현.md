# OpenDART 실데이터 연동 초기구현

- 작성시각: 2026-03-20
- 해결하고자 한 문제: 목업 데이터 위주로 동작하던 화면에 `research.md` 기준의 실제 공시/재무 데이터를 연결할 필요가 있었음.

## 진행 내용

- `research.md` 기준으로 `OpenDART 우선 연결` 방향을 적용함.
- `src/server/opendart.ts`를 추가해서 OpenDART 호출, 캐시, 공시/재무 정규화, 종목별 파생 지표 계산 로직을 구현함.
- `api/opendart/market-summary.ts`, `api/opendart/stock.ts`를 추가해서 Vercel 함수 진입점을 준비함.
- `vite.config.ts`에 로컬 개발용 `/api/opendart/*` 미들웨어를 추가함.
- `src/lib/live-data.ts`에 클라이언트 훅을 추가함.
- `HomePage`, `EmotionPage`, `StockPage`가 실제 API 응답을 읽어 표시하도록 수정함.
- `market.ts`는 종목 메타데이터와 기본 타입/호환 함수 중심으로 재정리함.
- 검증을 위해 `pnpm test`, `pnpm typecheck`, `pnpm lint`를 실행함.

## 해결된 것

- OpenDART 기반 실데이터 연동 구조가 로컬/Vercel 양쪽을 고려한 형태로 들어감.
- 홈/감정/종목 화면이 실데이터를 읽을 수 있는 상태가 됨.
- 실데이터가 없을 때도 기본 구조와 테스트는 유지되도록 호환 레이어를 둠.

## 아직 안 된 것

- 실제 응답 확인에는 `OPENDART_API_KEY`가 필요함.
- 뉴스/시세 무료 공개 서비스 소스는 아직 추가하지 않음.
- 감정 점수는 현재 공시/재무 기반 해석이어서 이후 고도화 여지가 있음.
