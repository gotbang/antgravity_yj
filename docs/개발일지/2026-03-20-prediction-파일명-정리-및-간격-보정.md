# prediction 파일명 정리 및 간격 보정

- 작성시각: 2026-03-20
- 해결하고자 한 문제: `predition.png` 오타 파일명을 정리하고, 현재 구현 화면의 간격을 Figma 메타데이터 기준에 더 가깝게 미세 조정해야 했음.

## 진행 내용

- `design/predition.png`를 `design/prediction.png`로 이름 변경함.
- 관련 문서의 파일 경로와 설명을 `prediction.png` 기준으로 모두 수정함.
- 상단 패널 패딩, 탭 높이, 제목 크기, 카드 반경, 홈 히스토리 행 배치, 검색창과 토글 버튼 간격을 조정함.
- 변경 후 `pnpm test`, `pnpm typecheck`, `pnpm lint`를 다시 실행함.

## 해결된 것

- 이미지 파일명이 일관되게 정리됨.
- 홈/종목/감정 화면이 Figma 원본 간격에 더 가깝게 보정됨.
- 검증 명령이 모두 통과함.

## 아직 안 된 것

- 실제 브라우저 스크린샷 대조 기반의 픽셀 단위 보정은 추가로 할 수 있음.
