# Ant Gravity MVP 초기구현

- 작성시각: 2026-03-20 20:22:03

## 해결하고자 한 문제

공개된 Figma site와 비슷한 흐름의 투자 인사이트 웹앱을 빈 워크스페이스에서 바로 구현해야 했고, 현재 세션에서는 Figma MCP를 직접 읽지 못하는 제약이 있었다.

## 진행 내용

- 공개된 Figma site 번들과 JSON을 직접 분석해서 정보 구조를 추출했어.
- React + Vite + TypeScript 프로젝트를 새로 세팅했어.
- 홈, 종목 분석, 시장 감정 3개 라우트를 구현했어.
- 관심 종목을 `localStorage`에 저장하는 상태를 붙였어.
- TDD 흐름으로 핵심 데이터 유틸 테스트를 먼저 만들고 통과시켰어.
- `lint`, `typecheck`, `test`, `build`까지 통과시켰어.

## 해결된 것

- 빈 프로젝트에서 실행 가능한 MVP를 만들었어.
- 공개 레퍼런스와 유사한 정보 구조를 화면으로 재현했어.
- 기본 샘플 데이터 기반 탐색과 관심 종목 관리가 가능해졌어.
- 프로젝트 문서 구조를 `docs`, `spec` 기준으로 만들었어.

## 아직 안 된 것

- Figma MCP를 통해 원본 노드 단위 디자인 컨텍스트를 읽지는 못했어.
- 실제 실시간 시세/API 연결은 아직 없어.
- 세부 인터랙션과 미세한 디자인 튜닝은 다음 단계에서 더 올려야 해.
