# OpenDART 로컬 키 주입 및 실응답 검증

- 작성시각: 2026-03-20 23:41:41 +09:00
- 해결하고자 한 문제: OpenDART 실데이터 연동 구조는 이미 있었지만 실제 키가 없어서 응답 검증이 안 된 상태였음.

## 진행 내용

- 로컬 전용 환경 파일에 OpenDART 키를 주입함.
- 키 값은 Git 추적 대상이 아닌 `.env.local`에만 저장함.
- OpenDART 회사 정보 API와 공시 목록 API를 직접 호출해서 실제 응답을 검증함.
- 삼성전자 기준으로 회사 정보와 최근 공시 목록이 `status: 000`으로 정상 응답하는 것을 확인함.
- 샌드박스 환경에서 Vite dev 서버는 `spawn EPERM`으로 실행되지 않아 화면 단 실응답 확인은 보류함.

## 해결된 것

- 로컬 키 주입 완료
- OpenDART 키 유효성 확인 완료
- 실제 외부 API 응답 확인 완료

## 아직 안 된 것

- 로컬 dev 서버에서 `/api/opendart/market-summary`, `/api/opendart/stock`를 거쳐 화면까지 치환되는 최종 확인
- 사용자 환경에서 `pnpm run dev` 기준의 실제 UI 검증
