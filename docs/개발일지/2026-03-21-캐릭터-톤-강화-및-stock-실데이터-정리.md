# 캐릭터 톤 강화 및 stock 실데이터 정리

- 작성시각: 2026-03-21 00:06:17 +09:00
- 해결하고자 한 문제: 전체 페이지의 캐릭터/폰트/분위기를 더 귀엽고 둥근 방향으로 맞추고, 종목분석 화면에서 실시간 가격처럼 보이는 부정확한 숫자 인상을 줄이며 실제 OpenDART 기반 데이터가 먼저 보이게 해야 했음.

## 진행 내용

- 헤드라인 폰트 스택과 브랜드 배지 스타일을 더 둥글고 친근한 톤으로 조정함.
- 캐릭터 배지와 상단 헤더 배경의 그림책 같은 분위기를 강화함.
- `StockPrediction` 타입에 최근 공시일, 최근 공시명, 공시 건수, 매출액, 영업이익, 부채비율 요약 필드를 추가함.
- OpenDART 서버 변환부에서 실제 공시/재무 응답을 위 필드로 매핑하도록 수정함.
- 종목분석 리스트 카드가 `실제 공시일 + 최근 공시명 + 매출액 + 영업이익 + 부채비율`을 우선 보여주도록 구조를 변경함.
- stock 화면 상단에 `실제 OpenDART 공시/재무 데이터` 안내와 `실시간 가격 미연동` 안내를 추가함.

## 해결된 것

- 전체 페이지가 기존보다 더 둥글고 친근한 폰트/캐릭터 톤에 가까워짐.
- 종목분석에서 실제 OpenDART 기반 데이터가 전면에 노출됨.
- 가격/시세는 아직 별도 연동 전이라는 사실을 명확히 보여주게 됨.
- `pnpm test -- --run`, `pnpm typecheck`, `pnpm lint`를 모두 통과함.

## 아직 안 된 것

- 실시간 가격은 OpenDART만으로는 제공되지 않아서 별도 시세 API 연동이 필요함.
- 첨부 레퍼런스와 완전히 동일한 캐릭터 사용은 원본 에셋 확보가 필요함.
