# 헤더 캐릭터 및 홈 바로가기 아이콘 실화면 보정

- 작성시각: 2026-03-21 01:12:00 +09:00
- 해결하고자 한 문제: 실제 localhost 화면 기준으로 헤더가 레퍼런스보다 딱딱하게 보였고, 브랜드 아이콘과 홈 바로가기 3개가 업로드한 이미지의 캐릭터/일러스트 톤과 다르게 보여서 실화면 기준 재보정이 필요했음.

## 진행 내용

- headless Chrome 캡처로 `http://localhost:5174/`, `http://localhost:5174/emotion` 화면을 직접 확인함.
- 공통 헤더의 기존 심볼 아이콘을 제거하고 업로드한 레퍼런스를 닮은 원형 개미 캐릭터 일러스트 SVG로 교체함.
- 홈의 `오늘 판단`, `종목 분석`, `개미의 일기` 카드 아이콘도 각각 일러스트형 SVG로 교체함.
- 헤더 배지 크기, 제목 타이포, 부제 크기, 네비게이션 칩 높이와 폰트 크기를 조정해서 더 부드럽고 안정적으로 보이게 다듬음.
- 빠른 카드 아이콘 박스 크기와 카드 라운드 값을 조정해서 레퍼런스의 둥글고 장난감 같은 인상에 더 가깝게 맞춤.

## 해결된 것

- 모든 페이지 공통 헤더가 기존 심볼보다 훨씬 캐릭터 중심으로 보이게 됨.
- 홈의 3개 바로가기 카드가 이모지 대신 레퍼런스 톤의 일러스트형 아이콘으로 통일됨.
- 실제 캡처 기준으로 헤더와 감정 페이지 상단이 자연스럽게 보이는 것까지 확인함.
- `pnpm test`, `pnpm typecheck`, `pnpm lint`를 다시 모두 통과함.

## 아직 안 된 것

- 사용자가 올린 원본 캐릭터 이미지를 그대로 자산화한 것은 아니고, 그 톤을 반영한 SVG 일러스트로 재구성한 상태임.
- 필요하면 이후 단계에서 실제 PNG/SVG 에셋을 따로 받아서 1:1에 더 가깝게 치환할 수 있음.
