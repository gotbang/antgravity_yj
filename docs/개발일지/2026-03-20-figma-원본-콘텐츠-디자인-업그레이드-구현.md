# Figma 원본 콘텐츠 + 업그레이드 디자인 구현

- 작성시각: 2026-03-20
- 해결하고자 한 문제: Figma 원본의 정보 구조를 유지하면서 `design/` 스크린샷 톤으로 실제 화면을 업그레이드해야 했음.

## 진행 내용

- Figma 메타데이터와 디자인 컨텍스트를 읽어서 홈 화면과 시장 감정 화면의 원본 섹션 구성을 다시 확인함.
- 기존 다크 데스크톱형 레이아웃을 모바일 퍼스트 카드 레이아웃으로 재구성함.
- `/`, `/stock`, `/stock/:symbol`, `/emotion`, `/diary`, `/journal` 라우트를 다시 정리함.
- 홈 화면은 `오늘의 주식 시장 방향`, `내 관심종목`, `모델 신뢰도`, `예측 히스토리` 구조로 재구현함.
- 시장 감정 화면은 `오늘의 시장 감정`, `투자자 심리 지표`, `뉴스 감정 분석` 구조로 재구현함.
- 종목 분석/예측, 일기, 저널 화면을 스크린샷 톤에 맞춰 새로 구성함.
- 테스트를 추가하고 `pnpm test`, `pnpm typecheck`, `pnpm lint`를 모두 통과시킴.

## 해결된 것

- Figma 원본 콘텐츠를 유지한 홈/감정 화면 구조가 실제 코드에 반영됨.
- 디자인 스크린샷 기준의 파스텔 카드, 네이비 헤더, 큰 라운드, 모바일 퍼스트 톤이 전역 스타일에 반영됨.
- 일기/저널 흐름과 종목 분석/예측 흐름이 한 프로젝트 안에서 공존하도록 라우트가 정리됨.

## 아직 안 된 것

- 실제 Figma 모든 세부 프레임을 1:1로 완전히 대조한 상태는 아님.
- 원격 에셋 아이콘은 직접 쓰지 않고 현재 코드베이스 스타일로 해석해서 재구성함.
- `prediction.png` 파일명 정리와 관련 문서 반영까지 완료함.
