# index.css 토큰화 및 디자인 프리뷰 생성

- 작성시각: 2026-03-21 00:22:54 +09:00
- 해결하고자 한 문제: 업로드한 `MASTER.md`, `design-tokens.md` 규칙을 실제 CSS 구조에 반영하고, 한 번에 검토 가능한 정적 디자인 프리뷰 페이지가 필요했음.

## 진행 내용

- `src/index.css` 상단에 컬러, 라운드, 섀도우, 간격, 폰트 토큰을 `:root`와 `.dark` 구조로 정리함.
- 기존 공통 컴포넌트 일부를 토큰 기반 변수로 치환해서 이후 유지보수 시 직접 색상/값을 덜 만지도록 정리함.
- `public/design-system-preview.html`을 생성해서 팔레트, 타이포, 버튼, 배지, 입력창, 라운드, 섀도우, 대표 화면 프리뷰를 한 번에 볼 수 있게 함.
- 기존 라우트 렌더가 깨지지 않도록 테스트/타입체크/린트를 재실행함.

## 해결된 것

- `src/index.css` 토큰 구조 정리
- 디자인 시스템 정적 프리뷰 페이지 생성
- `pnpm test -- --run`, `pnpm typecheck`, `pnpm lint` 모두 통과

## 아직 안 된 것

- 실제 shadcn/ui 토큰 파일로 완전 분리
- 프리뷰 페이지를 앱 안 라우트로 연결하는 작업
