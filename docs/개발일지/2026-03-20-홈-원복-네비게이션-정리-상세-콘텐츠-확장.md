# 홈 원복 네비게이션 정리 상세 콘텐츠 확장

- 작성시각: 2026-03-20 23:35:25 +09:00
- 해결하고자 한 문제: 공개 Figma site 기준의 정보 구조와 현재 구현이 어긋나 있었고, 홈 화면이 시장 감정 중심으로 밀려나며 종목 상세도 핵심 콘텐츠가 빠져 있었음.

## 진행 내용

- 홈 화면을 `오늘의 주식 시장 방향`, `내 관심종목`, `모델 신뢰도`, `예측 히스토리` 기준으로 복구함.
- 메인 네비게이션과 홈 바로가기에서 `개미의 일기`를 `/emotion` 흐름으로 다시 연결함.
- 종목 상세 화면에 `판단 근거`, `리스크 평가`, `참고 전략 가이드`, `데이터 믹스`, `AI 예측 따랐다면?` 섹션을 추가함.
- API 키가 없을 때도 샘플 콘텐츠가 먼저 보이도록 샘플 시장 데이터와 샘플 종목 상세 데이터를 정리하고 훅 fallback을 추가함.
- 공개 사이트의 텍스트/기능 구조를 반영하면서 디자인 톤은 `design/` 폴더 스크린샷 기준의 카드형 스타일을 유지하도록 CSS를 보강함.
- 라우트 테스트를 현재 구조에 맞게 업데이트함.

## 해결된 것

- 홈 정보 구조가 공개 기준에 맞게 복구됨.
- 메인 흐름 기준에서 `/emotion`이 다시 `개미의 일기` 역할을 담당하게 됨.
- 종목 상세에서 빠져 있던 분석 근거와 데이터 섹션이 복구됨.
- `OPENDART_API_KEY`가 없어도 샘플 데이터가 먼저 보이기 때문에 화면 비교가 쉬워짐.
- `pnpm test -- --run`, `pnpm typecheck`, `pnpm lint`를 모두 통과함.

## 아직 안 된 것

- 로컬 Playwright 브라우저 실행 오류로 최종 시각 검증은 완료하지 못함.
- 실제 OpenDART 키가 주입된 상태에서 실데이터 섹션이 새 구조와 자연스럽게 맞물리는지 추가 확인이 필요함.
- `/diary`, `/journal`을 계속 유지할지, 어느 수준까지 보조 흐름으로 둘지 최종 결정은 남아 있음.
