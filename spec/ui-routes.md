# UI / Route 계약

## 라우트

### `/`

- 목적: 시장 전체 방향을 빠르게 요약해서 보여줌
- 주요 데이터:
  - 시장 상승/하락 확률
  - 예상 변동 범위
  - 시장 리스크 점수
  - 관심 종목 리스트

### `/stock`

- 목적: 종목 검색과 탐색
- 주요 데이터:
  - 인기 종목 리스트
  - 관심 종목 리스트
  - 검색 결과 리스트

### `/stock/:symbol`

- 목적: 단일 종목의 상세 분석
- 주요 데이터:
  - 상승/하락 확률
  - 리스크 점수
  - 전략 문구
  - 재무/컨센서스/보조 지표

### `/emotion`

- 목적: 시장 감정 및 뉴스 톤 해석
- 주요 데이터:
  - 시장 감정 헤드라인
  - 공포/탐욕 성격 지표
  - 뉴스 긍정/부정 비율
  - 추천 대응 전략

## 상태 계약

### 관심 종목

- 저장 위치: 브라우저 `localStorage`
- 키: `ant-gravity.watchlist`
- 저장 형식: `string[]`
- 예시:

```json
["삼성전자", "NAVER"]
```

## 제약

- 서버 저장 없음
- 사용자 위치 정보 없음
- 민감 개인정보 없음
