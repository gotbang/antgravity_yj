# Ant Gravity 디자인 업그레이드 기준서

## 목적

- 이 문서는 현재 MVP를 `design/` 폴더의 스크린샷 기준으로 1:1에 가깝게 업그레이드하기 위한 단일 기준 문서다.
- 구현 우선순위는 기존 UI 유지가 아니라 스크린샷 기준의 화면 구조, 정보 위계, 색감, 간격, 컴포넌트 모양을 최대한 동일하게 맞추는 것이다.

## 기준 이미지

- `design/dashboard.png`
- `design/analysis.png`
- `design/prediction.png`
- `design/diary.png`
- `design/journal.png`

참고:
- `prediction.png`를 예측 상세 기준 이미지로 사용한다.

## 현재 `/emotion` 라우트 정리

- 기존 `/emotion`은 `시장 감정` 중심의 별도 대시보드였다.
- 핵심 구성은 `시장 심리 헤드라인`, `공포&탐욕 지수`, `수급 흐름`, `뉴스 감정 비율`, `추천 대응 전략`, `관심 종목 감정 메모`였다.
- 이번 디자인 기준에서는 이 화면을 유지하지 않는다.
- 디자인 기준상 감정 흐름은 `dashboard.png`의 시장 감정 카드 일부와 `diary.png`, `journal.png`의 일기/저널 흐름으로 재편된다.

## 라우트 매핑 가정

- `/` -> `dashboard.png`
- `/stock` -> `analysis.png`
- `/stock/:symbol` -> `prediction.png`
- `/diary` -> `diary.png`
- `/journal` -> `journal.png`

가정:
- 실제 구현 중 라우트 이름은 조금 달라질 수 있지만, 화면 책임은 위 매핑을 기준으로 유지한다.
- 디자인 1:1 반영이 우선이므로 기존 `/emotion`은 제거하거나 일기 흐름으로 대체한다.

## 전체 디자인 시스템

### 제품 톤

- 분위기는 `귀엽고 친근한 투자 도우미`다.
- 금융 앱처럼 차갑고 딱딱하기보다, 캐릭터와 파스텔 카드로 진입 장벽을 낮춘다.
- 차트보다 해석 카드와 감정 표현이 먼저 보이도록 구성한다.

### 레이아웃

- 모바일 퍼스트 고정 폭 카드 레이아웃 기준으로 설계한다.
- 기준 폭은 약 `390px` 전후의 휴대폰 화면이다.
- 전체 배경은 밝은 화이트 계열이고, 주요 정보는 떠 있는 카드처럼 보이게 한다.
- 섹션 사이 여백은 넉넉하게 두고, 카드 내부는 촘촘하지 않게 구성한다.

### 색상 팔레트

| 이름 | Hex | 역할 |
| --- | --- | --- |
| Navy Header | `#1F2B52` | 상단 브랜드 헤더, 탭 바, 강조 배경 |
| Soft Sky | `#CDEBFF` | 메인 감정 카드, 홈 첫 진입 카드 |
| Butter Yellow | `#FFE68A` | 종목 분석 카드, 주의/안내 배경, 지혜 카드 |
| Soft Pink | `#FFC7D7` | 일기 카드, 공포 영역, 하락 계열 강조 |
| Mint Green | `#CFF5D9` | 전략 카드, 상승 확률, 긍정 상태 배지 |
| Lavender Mist | `#E8E2FF` | 보조 배지, 선택 전 날짜/칩 계열 |
| Coral Red | `#F28B8B` | 하락 확률, 매도 태그, 위험 강조 |
| Neutral Text | `#111827` | 본문 텍스트 |
| Muted Text | `#6B7280` | 보조 설명, 시간, 서브카피 |
| Card White | `#FFFFFF` | 카드 기본 배경 |

주의:
- 정확한 수치보다 화면상 체감이 더 중요하다.
- 모든 카드에 진한 채도를 쓰지 말고, 파스텔 톤을 유지한다.

### 타이포그래피

- 전체 인상은 둥글고 친근한 산세리프다.
- 제목은 굵고 크며, 카드 안 핵심 지표는 한 번에 읽히게 크게 보여준다.
- 보조 설명은 회색으로 처리하되 대비가 너무 약해지지 않게 한다.
- 영문 브랜드명은 볼드, 한글 서브카피는 중간 굵기 정도로 둔다.

### 형태

- 최상단 브랜드 헤더는 큰 라운드 직사각형이다.
- 주요 카드 라운드는 `20px~28px` 수준의 둥근 모서리 느낌을 유지한다.
- 배지, 확률 바, 날짜 칩은 완전한 pill 형태를 적극 사용한다.
- 입력창도 직사각형보다 `두꺼운 라운드 아웃라인`이 있는 부드러운 형태로 맞춘다.

### 그림자와 깊이

- 그림자는 강하지 않다.
- 아래로 길게 떨어지는 그림자보다, 흐린 보라/회색 계열의 소프트 섀도우를 쓴다.
- 카드가 배경 위에 살짝 떠 보이는 정도만 유지한다.

### 아이콘과 일러스트

- 아이콘은 친근하고 작은 그림책 같은 느낌이어야 한다.
- 금융 의미 전달보다 정서 표현이 더 중요하다.
- 이모지, 캐릭터, 작은 일러스트를 카드 포인트로 사용한다.

## 화면별 상세 기준

### 1. 대시보드 `dashboard.png`

구성:
- 상단 네이비 브랜드 헤더
- 3개 진입 카드
- `오늘의 시장 감정` 섹션
- 시장 분위기 요약 카드
- 추천 대응 전략 목록

상세 규칙:
- 브랜드 헤더 안에는 원형 개미 프로필 이미지와 `Ant Gravity` 로고 텍스트를 둔다.
- 바로 아래 3개 진입 카드는 `오늘 판단`, `종목 분석`, `개미의 일기` 순서다.
- 각 카드 색상은 `하늘색`, `노란색`, `분홍색`으로 고정한다.
- 시장 감정 섹션은 큰 하늘색 카드와 우측 분홍색 보조 카드의 조합으로 보이게 한다.
- `공포 & 탐욕 지수`는 별도 강조 카드로 분리한다.
- `시장 분위기` 카드는 노란 배경 박스로 처리한다.
- `추천 대응 전략`은 연한 민트 배경의 pill 목록 3개 이상으로 보이게 한다.

### 2. 종목 분석 `analysis.png`

구성:
- 상단 네이비 헤더
- `종목 검색` 타이틀
- 둥근 검색 입력창
- `인기 종목 / 관심 종목` 2분할 탭
- 종목 카드 리스트
- 하단 도움말 배너

상세 규칙:
- 검색 입력창은 흰 배경 + 두꺼운 하늘색 테두리로 처리한다.
- 탭은 전체가 노란 컨테이너처럼 보이고, 선택된 탭은 흰 배경 카드처럼 튀어나오게 한다.
- 각 종목 행은 흰 카드 위에 `종목명`, `상승/중립/하락 배지`, `상승 확률`, `리스크 점수`를 가로 배치한다.
- 행 간 간격은 넉넉하고, 정보는 한눈에 읽히게 좌우 정렬한다.
- 상태 배지는 초록, 회색, 분홍 계열로 구분한다.
- 리스트 하단에는 연노랑 배경의 짧은 안내 배너를 둔다.

### 3. 예측 상세 `prediction.png`

구성:
- 상단 네이비 헤더
- 헤더 아래 3개 상단 탭
- `오늘의 예측` 메인 카드
- `예상 변동 범위`, `시장 리스크 점수` 요약 카드
- `모델 신뢰도` 요약 카드 2개
- `예측 히스토리` 리스트

상세 규칙:
- 헤더 바로 아래 탭은 네이비 바 위에 놓인 텍스트 탭처럼 보여야 한다.
- 메인 예측 카드는 화면에서 가장 큰 흰 카드로 둔다.
- 카드 중심에는 상승/하락 방향 아이콘과 큰 예측 결과 텍스트를 둔다.
- 상승/하락 확률은 빨강/초록 이중 progress bar 느낌으로 표현한다.
- 보조 수치 카드는 2열 그리드로 맞춘다.
- 히스토리는 민트와 분홍 배경 행으로 결과 성공/실패를 구분한다.

### 4. 일기 목록 `diary.png`

구성:
- 상단 네이비 헤더
- 가로 스크롤 날짜 칩
- `오늘의 일기` 카드
- `커뮤니티 다이어리` 카드 리스트
- 우하단 `Write My Diary` 플로팅 버튼

상세 규칙:
- 헤더는 기존 대시보드보다 더 세로로 길고, `Ant's Diary` 제목이 강조된다.
- 날짜 칩은 동그랗고 파스텔 색이 섞여 있어야 한다.
- 선택된 날짜는 노란색 계열 강조 칩으로 보이게 한다.
- 오늘의 일기 카드는 단일 요약형이고, 커뮤니티 다이어리는 여러 개의 흰 카드가 세로로 쌓이는 구조다.
- 카드에는 캐릭터 아바타, 이름, 시간, 짧은 메시지가 들어간다.
- 하단 FAB는 보라 계열 원형 버튼과 짧은 라벨이 함께 있는 구조를 유지한다.

### 5. 개인 저널 `journal.png`

구성:
- 상단 앱 바
- 월 선택 드롭다운과 이전/다음 화살표
- 월간 감정 그래프 카드
- 매매 일지 카드 리스트
- `개미의 지혜` 말풍선 카드
- 하단 탭 네비게이션

상세 규칙:
- 이 화면은 다른 화면보다 앱 네이티브 화면처럼 보여야 한다.
- 그래프 카드는 흰 배경에 연한 하늘색 area chart 형태다.
- 감정 레벨은 텍스트와 이모지로 함께 표시한다.
- 매매 일지 카드는 종목명, 매수/매도 pill, 짧은 메모, 수익 금액으로 구성한다.
- 수익 금액은 부호와 숫자가 눈에 띄게 정렬되어야 한다.
- `개미의 지혜`는 노란 말풍선 카드로 처리한다.
- 하단 탭 네비게이션은 아이콘 4개 구조로 구현한다.

## 공통 인터랙션 기준

- 탭은 선택 상태가 명확해야 한다.
- 탭 전환은 구조를 크게 바꾸지 말고, 내용 패널만 부드럽게 바뀌는 느낌이 좋다.
- hover 중심이 아니라 mobile tap 중심으로 설계한다.
- 모든 리스트와 카드 영역은 손가락으로 누르기 쉬운 높이를 유지한다.
- 스크롤이 필요한 화면은 하단 safe area를 고려한다.

## 구현 우선순위

1. 화면 구조와 카드 위계 일치
2. 색상과 라운드, 그림자 일치
3. 탭, 배지, 입력창 같은 반복 컴포넌트 일치
4. 현재 기능을 새 디자인 구조 안으로 재배치

## 비구현 방침

- 기존 `/emotion` 화면의 섹션 구조를 그대로 유지하지 않는다.
- 데스크톱 우선 레이아웃으로 확장하지 않는다.
- 불필요한 설정 옵션이나 테마 전환 기능을 추가하지 않는다.
- 실제 디자인에 없는 복잡한 차트, 테이블, 필터는 추가하지 않는다.

## 구현 체크리스트

- `dashboard`, `analysis`, `prediction`, `diary`, `journal` 5개 기준 화면을 모두 반영
- 탭 UI를 이미지 기준 그대로 구현
- 공통 헤더의 네이비 톤과 라운드 일관성 유지
- 파스텔 카드 팔레트 일관성 유지
- 상태 배지, 확률 바, 리스트 카드 스타일 통일
- 기존 `emotion` 흐름은 일기/저널 흐름으로 대체
